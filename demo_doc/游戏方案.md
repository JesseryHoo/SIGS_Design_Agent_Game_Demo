# SIGS AR寻宝 - 游戏化设计方案

## 🎮 核心游戏概念

### 游戏世界观

**背景设定**：
```
2050年，清华SIGS被神秘的"时空裂隙"影响，校园各处散落着"记忆碎片"。
玩家作为"时空修复者"的任务是收集这些碎片，恢复校园的历史记忆，
同时通过AI技术预览校园未来，为"校园改造计划"提供创意。
```

**玩家身份**：
```
身份：时空修复者（Time Restorer）
使命：收集记忆碎片 → 解锁历史 → 创造未来
```

## 🎭 六大游戏机制

### 机制1：剧情叙事系统 ⭐⭐⭐

#### 主线剧情设计

**第一章：初入时空（新手引导）**
```
剧情：
- 开场动画：神秘教授发来消息
- "校园出现了异常...记忆碎片正在消失..."
- "只有你能拯救这段历史"

任务流程：
任务1.1: 前往信息大楼 → 发现第一个碎片
任务1.2: 学习如何收集碎片
任务1.3: 收集3个普通碎片
任务1.4: 与教授对话，了解真相

奖励：解锁"初级修复者"称号 + 50积分
```

**第二章：失落的记忆**
```
剧情：
- 发现碎片中隐藏着建校历史
- 每个建筑都有不为人知的故事
- 需要收集特定系列碎片拼凑真相

任务：
- 收集"建校系列"碎片（5个）
- 解答历史谜题
- 拼凑完整历史故事

奖励：解锁"历史学者"称号 + 历史图鉴
```

**第三章：未来预览**
```
剧情：
- AI预测显示校园需要改造
- 收集"创新火花"解锁AI预测
- 为校园改造提供建议

任务：
- 收集"创新系列"碎片
- 使用AI生成改造方案
- 提交到"未来提案库"

奖励：解锁"未来设计师"称号 + AI使用权
```

**第四章：时空守护者**
```
剧情：
- 发现破坏者正在窃取碎片
- 需要联合其他玩家阻止
- 最终决战

任务：
- 组队协作任务
- 收集传说级碎片
- 击败破坏者（解谜挑战）

奖励：解锁"时空守护者"称号 + 限定皮肤
```

```

## 🎨 视觉与音效设计

### 稀有度视觉区分

```
普通品质：
- 边框：灰色
- 光效：弱白色光芒
- 音效：轻柔"叮"

稀有品质：
- 边框：蓝色
- 光效：蓝色光芒
- 音效：清脆"叮~"

史诗品质：
- 边框：紫色
- 光效：紫色粒子
- 音效："叮~✨"

传说品质：
- 边框：金色
- 光效：金色爆炸效果
- 音效："✨叮✨~" + 震动
```

### 特殊效果

**收集成功**：
```
- 物品从地上飞起
- 旋转放大
- 粒子爆炸
- 音效 + 震动
- 显示获得的积分
```

**升级**：
```
- 屏幕闪光
- 等级数字滚动
- "LEVEL UP!"动画
- 特殊音效
- 震动模式：[100, 50, 100, 50, 200]
