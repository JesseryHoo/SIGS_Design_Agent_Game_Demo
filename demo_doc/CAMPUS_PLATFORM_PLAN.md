# 清华大学深圳国际研究生院校园民众参与平台 - 完整实施计划

## 📋 项目概述

### 项目名称
**"清华SIGS校园探索者" - 沉浸式校园互动平台**

### 核心愿景
打造一个融合虚拟漫游、游戏化学习、现实激励的校园参与平台，让师生、访客通过互动体验深度了解校园文化、参与校园建设、增强归属感。

---

## 🎯 项目目标

### 短期目标
- 创建可玩性强的校园3D探索游戏
- 建立线上积分与线下奖励的闭环机制
- 收集用户行为数据，优化校园体验

### 长期目标
- 形成校园文化IP和品牌效应
- 为校园规划和管理提供数据支持
- 构建可持续的校园社区生态

---

## 🌟 核心概念与主题升华

### 主概念：三重融合
```
虚拟探索 + 游戏化学习 + 现实激励 = 深度校园参与
```

### 主题升华方向

#### 1️⃣ **校园文化传播大使**
- **玩法核心**：玩家扮演"校园文化探索者"，通过解密校园历史、建筑故事、科研成就来解锁关卡
- **社会价值**：让每位参与者成为校园文化的传播者，增强文化认同感和传承意识

#### 2️⃣ **校园共建参与者**
- **玩法核心**：在探索中发现问题、提出建议，优秀建议可转化为实际校园改进
- **社会价值**：形成"发现问题-提出方案-实际改进"的民主参与闭环

#### 3️⃣ **科研创新启蒙者**
- **玩法核心**：打卡各实验室、科研中心，了解前沿研究，触发科研灵感
- **社会价值**：促进跨学科交流，激发创新思维，培养科研兴趣

#### 4️⃣ **可持续发展践行者**
- **玩法核心**：通过完成环保任务（如垃圾分类、节能打卡）获得额外奖励
- **社会价值**：培养环保意识，推动绿色校园建设

## 🎮 玩法设计

### 核心玩法体系

#### A. 探索打卡系统 📍
```
基础玩法：自由探索 → 发现地标 → 打卡解锁 → 获得积分
```

**具体设计：**
- **地标点分类**：
  - 历史文化类：校史馆、名人雕像、历史建筑
  - 科研教学类：各学院楼、实验室、图书馆
  - 生活服务类：食堂、运动场、宿舍区
  - 自然景观类：湖泊、花园、观景台

- **打卡机制**：
  - 物理到达：GPS定位+AR扫描验证
  - 虚拟到达：3D场景中角色移动到目标点
  - 知识问答：每个地标3-5道相关题目
  - 影像记录：上传照片/视频完成打卡

#### B. 闯关任务系统 🎯
```
主线任务：校园探索历程
支线任务：专题深度体验
日常任务：持续参与激励
```

**主线任务设计（7天新手引导）：**

| 天数 | 任务主题 | 任务内容 | 奖励 |
|------|----------|----------|------|
| Day 1 | 初识校园 | 打卡3个主要建筑，完成新生指南 | 校徽虚拟徽章 |
| Day 2 | 历史探寻 | 探索校史馆，回答历史问题 | "历史探索者"称号 |
| Day 3 | 科研之旅 | 打卡3个实验室，了解研究方向 | 科研积分×2 |
| Day 4 | 绿色校园 | 完成3个环保小任务 | 环保卫士徽章 |
| Day 5 | 文化使者 | 参与/创建1个文化活动 | 文化贡献积分 |
| Day 6 | 校园建设者 | 提出1条校园改进建议 | 建设者勋章 |
| Day 7 | 终极挑战 | 综合知识测试，完成全地图70%探索 | "校园专家"称号+实体奖励 |

**支线任务系列：**
- 🔬 **科研先锋系列**：深度了解各实验室研究
- 🎨 **文化艺术系列**：探索艺术作品、文化活动
- 🌱 **绿色校园系列**：环保实践、低碳行动
- 🤝 **社区互动系列**：多人协作任务
- 📸 **创意拍摄系列**：校园摄影比赛

#### C. 积分奖励系统 💎

**积分获取：**
```
基础分：
- 地标打卡：10-50分（根据重要性和难度）
- 问答正确：5-20分/题
- 任务完成：50-200分/任务

额外奖励：
- 连续打卡：+10%/天（最高+50%）
- 首次发现：+100分
- 精彩分享：+30分
- 优质建议：+100分
```

**等级系统：**
```
Lv.1 新生探索者   0-500分
Lv.2 校园向导     500-2000分
Lv.3 文化使者     2000-5000分
Lv.4 科研伙伴     5000-10000分
Lv.5 校园专家     10000-20000分
Lv.6 荣誉校友     20000分+
```

**奖励兑换：**
```
虚拟奖励：
- 独特头像框、徽章
- 限定皮肤、装扮
- 特殊称号、头衔

实体奖励（积分兑换）：
- 500分：清华明信片套装
- 1000分：清华书签
- 2000分：清华笔记本
- 5000分：清华T恤
- 10000分：清华模型
- 20000分：限量文创套装
```

#### D. 社交互动系统 👥

**互动功能：**
- 好友系统：添加好友、查看进度、组队探索
- 排行榜：周榜、月榜、总榜、分类榜
- 协作任务：多人合作完成大型任务
- 分享功能：成就分享到社交媒体

## 🎨 3D场景建设方案

### 数据采集方案

#### 方案A：专业建模（高质量）
```
1. 获取建筑图纸和CAD文件
2. 无人机航拍获取地形数据
3. 激光扫描获取建筑精确数据
4. 专业建模师手工建模
5. UE5材质和光照处理

优点：质量最高，效果最好
缺点：成本高，周期长
```

#### 方案B：摄影测量（平衡方案）
```
1. 多角度拍摄建筑照片
2. 使用RealityCapture/ContextCapture生成模型
3. Blender进行优化和修整
4. 导入UE5进行后期处理

优点：成本适中，效果好
缺点：需要大量照片素材
```

#### 方案C：混合方案（推荐）
```
主要建筑：方案A（重点区域）
次要建筑：方案B（一般区域）
环境：使用UE5资产库快速搭建

优点：平衡质量和成本
缺点：需要多种技能配合
```

### 场景优化策略

```
1. LOD（细节层次）系统
   - LOD0: 近景高模
   - LOD1: 中景中模
   - LOD2: 远景低模

2. HLOD（分层LOD）
   - 合并远处静态物体
   - 减少Draw Call

3. 虚拟纹理
   - 支持超大纹理
   - 按需加载

4. 性能预算
   - 移动端：50万面以内
   - PC端：200万面以内
   - 目标帧率：60fps
```
